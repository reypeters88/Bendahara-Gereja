Install Aplikasi di HP Android & Tablet

Buka secara Fullscreen layaknya Aplikasi Resmi di layar utama
Bendahara Gereja Advent (GMAHK)
Sistem Arus Kas & POS Kuitansi Digital
Cara 1: Install Langsung (1-Klik via Browser HP)

Aplikasi ini didukung teknologi Progressive Web App (PWA). Klik tombol di bawah untuk memasang aplikasi langsung ke layar utama HP Android atau Tablet Anda:

💡 Panduan Manual Chrome / Edge di HP Android:
1. Ketuk ikon 3 titik di pojok kanan atas browser.
2. Pilih menu "Install aplikasi" (Install app) atau "Tambahkan ke layar utama" (Add to Home screen).
3. Ikon GMAHK akan muncul di HP Anda dan bisa dibuka offline tanpa mengetik alamat web lagi!
Cara 2: Membuat File Standalone (.APK / .AAB)

Jika Anda ingin membagikan aplikasi ini sebagai file BendaharaGMAHK.apk yang bisa diinstall langsung (misal dikirim lewat WhatsApp):

  1. Unggah folder aplikasi ini ke situs hosting atau GitHub Pages gratis.
  2. Buka alat resmi Microsoft gratis: PWABuilder.com
  3. Masukkan link web aplikasi Anda, lalu klik Start.
  4. Klik tombol Package for Android ➔ File .APK siap didownload dalam beberapa detik!
